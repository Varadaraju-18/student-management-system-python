# Student Management System (Python + SQLite)

This is a console-based Student Management System developed using Python and SQLite database.

## Features
- Add new student records
- View all student details
- Update student marks
- Delete student records
- Data stored using SQLite database

## Technologies Used
- Python
- SQLite

## How to Run
1. Make sure Python is installed.
2. Download or clone this repository.
3. Open terminal in project folder.
4. Run the command:
   python main.py

## Purpose
This project is created as an academic mini project to understand database connectivity and CRUD operations using Python.
